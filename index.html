    📍 찾는 방법:
    1. 연동할 구글 시트를 엽니다.
    2. 주소창의 URL을 확인합니다.

    1. Google Cloud Console 접속
    2. [API 및 서비스 > 라이브러리]에서 Google Sheets API '사용' 클릭
    3. [사용자 인증 정보 > 인증 정보 만들기]에서 API 키 선택
    4. 생성된 AIza...로 시작하는 키를 복사하세요.
    📍 배포 방법:

    2. 제공된 apps-script.gs 코드를 붙여넣고 저장
    3. [배포 > 새 배포] 클릭 (유형: 웹 앱)
    4. 액세스 권한을 [모든 사람(Anyone)]으로 설정 후 배포
